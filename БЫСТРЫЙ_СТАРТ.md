# Быстрый старт - парсер на GitHub Actions

## Шаг 1: Создать репозиторий на GitHub (1 минута)

1. Зайти на https://github.com/new
2. Имя репозитория: `che168-parser`
3. Сделать **Public** (бесплатно)
4. Нажать "Create repository"

## Шаг 2: Запушить код (2 минуты)

Выполнить команды в терминале:

```bash
cd "C:\Users\MSI\Desktop\Клауд\Курсор\Автосайты\Парсеры"
git remote add origin https://github.com/ВАШ_НИК/che168-parser.git
git branch -M main
git push -u origin main
```

## Шаг 3: Запустить парсер (1 минута)

1. Открыть https://github.com/ВАШ_НИК/che168-parser/actions
2. Нажать "Che168 Parser" → "Run workflow"
3. Выбрать ветку "main" → "Run workflow"
4. Ждать 1-2 минуты

## Результат

- Кликнуть на зеленый запуск в Actions
- Скачать артефакт `che168_result.json`
- Или смотреть JSON прямо в логе
