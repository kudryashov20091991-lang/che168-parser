1. Откройте сайт: https://2ip.ru/ping/ или https://ping-admin.ru/ping

2. Введите каждый поддомен по очереди:

1. Откройте командную строку (cmd)


## Шаг 4: Проверьте что IP разные

**Хороший результат:**
- 8-10 уникальных IP из 10 поддоменов
- Некоторые поддомены могут быть на одном IP (это нормально)

**Плохой результат:**
- 1-3 уникальных IP из 10 поддоменов
- Все поддомены на одном IP

**Что делать если IP одинаковые:**

1. Создайте поддомены с другими именами:
   ```
   pr1.best-japan-cars.ru
   pr2.best-japan-cars.ru
   px1.best-japan-cars.ru
   px2.best-japan-cars.ru
   ```

2. Иногда помогает пересоздание поддомена:
   - Удалите поддомен
   - Создайте заново через 5-10 минут
   - Проверьте IP снова
