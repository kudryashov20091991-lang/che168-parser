# СРОЧНО: СОЗДАТЬ ТОКЕН GITHUB (2 минуты)

## Шаг 1: Открыть страницу токена

**Скопировать и открыть в браузере:**
```
https://github.com/settings/tokens/new
```

## Шаг 2: Войти в GitHub

- **Email:** `kudryashov20091991@gmail.com`
- **Пароль:** `!1Vbkkbfhl _4`

## Шаг 3: Создать токен

1. **Token name:** `che168-parser`
2. **Expiration:** No expiration (или 90 дней)
3. **Permissions - отметить галочки:**
   - ✅ **repo** (все подпункты автоматически)
   - ✅ **workflow**

4. Нажать **"Generate token"** (внизу страницы)

## Шаг 4: Скопировать токен

- Появится токен вида: `ghp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`
- **Нажать кнопку Copy** (иконка копирования)
- **Сохранить в блокнот!** (больше не покажется)

## Шаг 5: Запустить настройку

1. Открыть файл: `SETUP_GITHUB.bat` (в этой же папке)
2. Вставить токен когда попросит
3. Нажать Enter
